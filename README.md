# Children's Games

A small offline-friendly collection of simple browser games for young kids. The project is a static launcher plus standalone game pages that can run locally by double-clicking, with gentle play, large touch targets, and no install step.

## Games

- Cooking
- Doll Dress Up
- Math Fun
- Spelling
- Tic Tac Toe
- Grocery Store

## Run Locally

Open `index.html` or `Dashboard.html` in a browser. No install, build, server, or command line is needed.

## Deploy To Vercel

Import the GitHub repo `dennisherreraETH/childrensgames` into Vercel. Use framework preset `Other`, leave the build command empty, and deploy the repo root as the output.

## Tech Note

This is vanilla HTML/CSS/JS with no dependencies, no build step, no network requests, and no browser storage.
